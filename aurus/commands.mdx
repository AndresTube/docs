## Commands

| Command | Description |
|---|---|
| `/au open menu_id [player]` | Opens a menu |
| `/au close [player]` | Closes the active menu |
| `/au reload` | Reloads all menu files and config |
| `/au debug` | Toggles debug mode |

Alias: `/aurus`

## Permissions

| Permission | Description | Default |
|---|---|---|
| `aurus.admin` | Access to all subcommands | op |

## Cursor configuration

```yaml
cursor:
  type: TEXT
  value: "●"
  size: 1.0
```

| Key | Type | Default | Description |
|---|---|---|---|
| `type` | string | `TEXT` | Cursor display type |
| `value` | string | `●` | The character or material name |
| `size` | double | `1.0` | Scale of the cursor |

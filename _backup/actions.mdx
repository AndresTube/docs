---
title: "Actions"
description: "Run commands, send messages, play sounds, and navigate menus"
---

Actions are strings that run top-to-bottom.

## Available actions

| Action | Example | Description |
|---|---|---|
| `[close]` | `[close]` | Closes the menu |
| `[player] cmd` | `[player] spawn` | Runs a command as the player |
| `[console] cmd` | `[console] give %player% diamond 1` | Runs a command from console |
| `[message] text` | `[message] Hello!` | Sends a message to the player |
| `[broadcast] text` | `[broadcast] %player% joined!` | Broadcasts to all players |
| `[openmenu] id` | `[openmenu] user_profile` | Opens another menu |
| `[sound] key, vol, pitch` | `[sound] entity.player.levelup, 1.0, 1.2` | Plays a sound |

## Example

```yaml
my_button:
  type: BUTTON
  text: "<aqua>Go to Spawn"
  x: 0.0
  y: 0.0
  size: 1.5
  actions:
    - "[close]"
    - "[player] spawn"
    - "[sound] entity.player.levelup, 1.0, 1.0"
    - "[message] <green>Teleported!"
```
